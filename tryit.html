Try out GNOME 3

GNOME 3 is under active development and will not be completed until April 2011, but you can try the development release with our live USB image.

Download for 32 bit systems
Download for 64 bit systems
To get it running:
  • sudo chmod a+x ./image-writer
  • Insert an empty USB stick
  • ./image-writer img_file.iso
  • Reboot your computer and run it!

You can also use the ISO to burn a DVD, or run in a virtual machine.
